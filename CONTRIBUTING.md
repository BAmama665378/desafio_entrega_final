# Guia de Colaboração - PortfolioHUB 🚀

Primeiramente, obrigado por demonstrar interesse em colaborar com o projeto PortfolioHUB! Este é um espaço centralizado para exibição de portfólios digitais integrado com práticas de segurança orientadas pela IA Google GEMINI.

Para garantir a organização, a segurança e a padronização do código, pedimos que todos os colaboradores sigam as diretrizes abaixo.

---

## 🛠️ Como Contribuir

### 1. Bifurcar o Projeto (Fork)
Antes de fazer qualquer alteração, faça um **Fork** deste repositório para a sua própria conta do GitHub.
